# 🎹 Melodify

Generate MIDI melodies from pictures

# install
```
git clone https://github.com/rodezee/melodify
cd melodify
pip install mido pillow --break-system-packages
chmod +x melodify

```

# use
Get yourself a picture and run:
```
./melodify picture.jpg

```

To see the custom options:
```
./melodify --help

```
